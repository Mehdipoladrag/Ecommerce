# Ecommerce

## Run and Install Project in Docker 
```
docker-compose up -d --build
```

## Create Container
```
docker-compose up -d 
```

## Remove Container
```
docker-compose down
```

## Create App
```
django-admin startapp appname
```

### Run Server 
```
python manage.py runserver
```

### DataBase Migrations
```
python manage.py migrate
```


### Create SuperUser
```
python manage.py createsuperuser
```
